# My Project

A Python project.

## Setup

1. Create a virtual environment:
   ```
   py -m venv venv
   ```
   or
   ```
   python -m venv venv
   ```

2. Activate it (PowerShell):
   ```
   .\venv\Scripts\Activate
   ```

3. Install dependencies (if any):
   ```
   pip install -r requirements.txt
   ```

## Run

```
python main.py
```
